# Introduction

**Risques Majeurs MCP** est un serveur [MCP](https://modelcontextprotocol.io/) (Model Context Protocol) qui permet à un agent IA d'interroger l'exposition aux risques majeurs pour n'importe quelle adresse en France.

## À quoi ça sert ?

Imaginez que vous discutez avec un assistant IA et que vous lui demandez :

> *"Quels sont les risques naturels autour du 20 avenue de Ségur à Paris ?"*

Grâce à ce serveur MCP, l'assistant peut :

1. **Géocoder l'adresse** pour obtenir les coordonnées GPS
2. **Interroger les API Géorisques** pour évaluer l'exposition aux différents risques
3. **Afficher une carte interactive** avec les zones de risques

Le tout de façon automatique, en exploitant les données ouvertes du Ministère de la Transition Écologique.

## Pourquoi MCP ?

Le [Model Context Protocol](https://modelcontextprotocol.io/) est un standard ouvert qui permet aux modèles de langage d'interagir avec des sources de données et des outils externes. Plutôt que d'intégrer les données directement dans le modèle, MCP fournit une interface standardisée pour que l'IA puisse :

- **Appeler des outils** (géocodage, évaluation des risques)
- **Récupérer des données structurées** (niveaux d'exposition, listes de risques)
- **Afficher des interfaces riches** (cartes interactives)

## Données ouvertes

Ce projet s'appuie entièrement sur des **API publiques et ouvertes** :

| Source | Usage |
|---|---|
| [Géorisques](https://www.georisques.gouv.fr/) | Données d'exposition aux risques naturels et technologiques |
| [Géoplateforme IGN](https://data.geopf.fr/) | Géocodage des adresses en France |
| [CARTO](https://carto.com/) | Tuiles de fond de carte |
