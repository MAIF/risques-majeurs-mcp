# Démos

Ces démos montrent, vidéo à l’appui, comment connecter le serveur **Risques Majeurs MCP** aux principales plateformes IA et l'utiliser pour interroger l'exposition aux risques d'une adresse en France.

## Démos disponibles

| Plateforme | Démo |
|---|---|
| **ChatGPT** (OpenAI) | [Démo — ajouter le serveur à ChatGPT](./chatgpt) |
| **Le Chat** (Mistral) | [Démo — ajouter le serveur à Le Chat](./mistral) |

Les deux démos suivent la même structure :

1. Lancer le serveur MCP en HTTP localement.
2. Exposer le serveur via un tunnel HTTPS public (ngrok ou cloudflared).
3. Ajouter un connecteur MCP dans le client cible.
4. Tester depuis une conversation.

Pour les autres plateformes (Claude Desktop, Claude Code, Codex CLI, Gemini CLI…), reportez-vous à la page [**Connecter un client IA**](../clients-mcp) qui détaille la configuration plateforme par plateforme.
