# Carte d'exposition aux risques

L'outil `carte_exposition_risques` combine l'évaluation de l'exposition aux risques avec une **carte interactive** affichant les couches de données géographiques. Il utilise le mecanisme de [MCP App](https://modelcontextprotocol.io/) pour afficher une application web embarquee dans le client MCP.

## Parametres d'entrée

| Parametre | Type | Obligatoire | Description |
|---|---|---|---|
| `longitude` | `number` | Oui | Longitude (EPSG:4326, entre -180 et 180) |
| `latitude` | `number` | Oui | Latitude (EPSG:4326, entre -90 et 90) |
| `risques` | `string[]` | Non | Liste des codes de risques à afficher (défaut: tous) |

:::info
Tous les risques ne sont pas disponibles sur la carte. L'outil `liste_risques` indique quels risques supportent la visualisation cartographique. Actuellement, `catnat` (catastrophes naturelles) n'a pas de couche cartographique.
:::

## Exemple d'appel

```json
{
  "longitude": 3.8767,
  "latitude": 43.6117,
  "risques": ["argiles", "inondations"]
}
```

## Réponse structurée

En plus de la carte, l'outil retourne les données structurées d'exposition pour chaque risque, exploitables par l'agent IA.

## La carte interactive

La carte est une application web basée sur [MapLibre GL](https://maplibre.org/) qui offre :

- **Fond de carte** — Tuiles CARTO en niveaux de gris
- **Marqueur de position** — Indique les coordonnées demandees
- **Couches de risques** — Superposition des données WMS de Géorisques ou GeoJSON
- **Contrôle des couches** — Cases à cocher pour afficher/masquer chaque risque
- **Légendes** — Légende interactive pour chaque couche affichée
- **Mode plein ecran** — Bascule entre vue inline et plein ecran (si supporte par le client)

![Carte d'exposition aux risques avec contrôle des couches](/img/screenshots/map_tool.png)

En cliquant sur le bouton **Légende** en bas à gauche, un panneau détaille les symboles et codes couleurs de chaque couche affichée :

![Carte d'exposition aux risques avec légende deployee](/img/screenshots/map_tool_legend.png)

### Types de couches

| Risque | Type de couche | Source |
|---|---|---|
| Argiles | WMS raster | Géorisques (`ALEARG`) |
| Mouvements de terrain | WMS raster | Géorisques (`MVT_LOCALISE`) |
| Cavités | WMS raster | Géorisques (`CAVITE_LOCALISEE`) |
| Inondations — TRI | WMS raster | Géorisques (`LIMITETRI_FXX`) |
| Inondations — PPR | WMS raster | Géorisques (`PPRN_INOND`) |
| ICPE Seveso | GeoJSON (points) | API Géorisques |
| Installations nucléaires | GeoJSON (points) | API Géorisques |
